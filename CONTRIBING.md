# Contributing to SoftvibesLab

Gracias por tu interes en contribuir a proyectos de SoftvibesLab.

## Como Contribuir

### 1. Encontrar un Proyecto

Busca proyectos en: https://github.com/softvibeslab

### 2. Fork y Clonar

```bash
git clone https://github.com/softvibeslab/[nombre-proyecto].git
```

### 3. Crear Rama de Feature

```bash
git checkout -b feature/[tu-feature]
```

### 4. Hacer Cambios

Sigue las directrices de:
- CODING_GUIDELINES.md
- GIT_WORKFLOW.md
- TESTING.md

### 5. Commits Siguientes las Directrices

Usa Conventional Commits:
```bash
git commit -m "feat(tipo): descripcion breve"
```

### 6. Push y Pull Request

```bash
git push origin feature/[tu-feature]
```

Crea PR en GitHub con:
- Titulo descriptivo
- Descripcion de cambios
- Referencia a issues relacionados
